# Sistema de Cadastro de Alunos

## Descrição

Este projeto é um sistema simples de cadastro de alunos desenvolvido em Java utilizando vetores para armazenamento dos dados. O programa permite cadastrar, listar, buscar e remover alunos através de um menu interativo executado no terminal.

## Funcionalidades

* Cadastro de até 10 alunos.
* Listagem de todos os alunos cadastrados.
* Busca de aluno pelo nome.
* Remoção de aluno pelo nome.
* Encerramento do sistema.

## Menu Principal
Ao executar o programa, o usuário terá acesso às seguintes opções:

===== SISTEMA DE CADASTRO DE ALUNOS =====
1 - Cadastrar aluno
2 - Listar alunos
3 - Buscar aluno pelo nome
4 - Remover aluno
5 - Sair

## Requisitos
* Java JDK 8 ou superior.
* Classe auxiliar `IO` para entrada e saída de dados.

## Como Executar

1. Compile o arquivo:
javac cadastroAluno.java

2. Execute o programa:
java cadastroAluno
